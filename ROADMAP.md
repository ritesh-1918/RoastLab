# RoastLab Roadmap

Public roadmap. Items roughly ordered by priority.

## Now (in progress)

- [x] Neon Postgres — audit persistence
- [x] Auto-email reports via Resend
- [x] Admin dashboard
- [x] Forgot password flow
- [x] Real-time profile editing
- [x] Telugu/Hindi meme stickers
- [x] Deep crawl for premium users

## Next

- [ ] **Comparison mode** — audit two URLs side by side
- [ ] **Re-audit** — click to re-audit a saved URL and see score diff
- [ ] **Shareable report link** — public URL for a specific audit
- [ ] **Comment on findings** — annotate roast items
- [ ] **Slack / Discord notifications** — post audit summary to a channel

## Later

- [ ] **Browser extension** — right-click any page → instant roast
- [ ] **Team accounts** — multiple users sharing one workspace
- [ ] **Zapier / n8n integration** — trigger audits via automation
- [ ] **White-label** — agency tier with custom branding on reports
- [ ] **Competitor benchmarking** — audit your site vs competitor
- [ ] **Weekly monitor** — scheduled re-audit with score change alerts

## Ideas (no commitment)

- Figma plugin — audit your design before shipping
- CLI — `npx roastlab audit https://example.com`
- VS Code extension — audit any HTML file
- API — programmatic access
